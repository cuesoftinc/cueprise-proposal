Unlike standard ERP permissions, MSN Energy's deployment implements a "Rank & Role" hierarchy with document-level invisibility.

File Invisibility

If a user's rank is lower than a document's "Sensitivity Level," the file is not just locked—it is invisible in the UI.

Approval Authority

Financial approval limits are tied directly to the user's rank in the HR module.

Admin Lockdown
